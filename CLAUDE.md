# Project Overview
Next.js 16 web application integrated with AWS Amplify Gen 2.
Features a frontend-heavy architecture with TypeScript, TailwindCSS 4, and React 19.

# Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS 4, PostCSS
- **Backend/Cloud**: AWS Amplify Gen 2, AWS CDK
- **Data**: SWR for fetching
- **UI Components**: Lucide React, Framer Motion, Recharts

# Key Directories
- `src/app`: App Router pages and layouts
- `src/components`: Reusable UI components
- `src/lib`: Utility functions and Amplify configuration
- `src/context`: React context providers
- `amplify`: AWS Amplify backend definition and config

# Build & Test Commands
- **Dev Server**: `npm run dev` (running with Turbopack)
- **Build**: `npm run build`
- **Lint**: `npm run lint`
